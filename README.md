# Шаблон для отправления преобразованных данных в чат Битрикс24
API Bitrix, вебхук, аггрегация, SQL-запрос в pd.DataFrame, преобразование, отправка в чат

- скрипт собирает нужные преобразованные данные из хранилища (SQL-запросы скрыты)
- после преобразования создаются те или иные строковые выражения, говорящие о динамике метрик
- создаётся структурированный текст еженедельной и месячной динамики метрик
- такой отчёт отправляется в нужные чаты Битрикс24
